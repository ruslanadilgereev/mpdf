# Changelog

All notable changes to the MPDF format and tools.

## [1.0.0] - 2026-03-18

### Added
- MPDF Format Specification v1.0
- Reference implementation in Python (`mpdf-create.py`)
- Validator tool (`mpdf-validate.py`)
- Python library (`pip install mpdf`)
- CLI tool with `create`, `validate`, and `view` commands
- Platform setup scripts for Windows, macOS, and Linux
- Example documents (minimal, academic paper, tutorial)
- GitHub Actions CI
- GitHub Pages landing page
